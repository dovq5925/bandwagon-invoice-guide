## 搬瓦工费用清单是什么？

搬瓦工的"费用清单"就是它的 Invoice（账单）。每次购买新套餐或续费时，系统会自动生成一张英文账单，记录这笔交易的具体内容。

**一张完整的搬瓦工 Invoice 包含以下信息：**

1. Invoice 编号（唯一单据编号）
2. Invoice To（账单抬头，也就是你的账户姓名/公司名）
3. 服务描述（购买的套餐名称、计费周期）
4. 金额明细（原价、折扣、实付金额）
5. 付款状态（Paid / Unpaid）
6. 交易时间

这就是搬瓦工费用清单的完整构成。没有增值税号，没有服务商的营业执照信息——这是海外服务商的普遍情况，不是搬瓦工特例。

## 搬瓦工费用清单的"抬头"怎么改

这是很多要报销的人会卡住的地方。

搬瓦工 Invoice 上有一个 **Invoice To** 字段，显示的是你账户里填写的名字或公司名称。如果你注册时随便填了个名字，这里就会显示那个名字，和你公司的抬头对不上。

好消息是：这个可以改。

在账户设置里修改你的姓名/公司信息之后，重新下载 PDF，抬头就会同步更新。不用联系客服，自助操作就行。

修改完之后再下一张 Invoice 给财务，报销通过率会高不少。

## 新购账单 vs 续费账单，有什么不同

这两种账单在逻辑上是一样的，都是 Invoice，格式相同。区别主要在时间和有效期上。

**新购账单：**
- 下单后立即生成
- 有效期约 3 天，超时自动取消
- 支付成功后状态变为 Paid

**续费账单：**
- 到期前 7 天自动生成（搬瓦工最近更新了这个规则，之前是到期当天才出账单）
- 有效期约十几天，但续到什么时候还是从账单生成日算起——所以拖着不付也没用，早点付就好
- 搬瓦工不会自动从支付宝或信用卡扣款，但如果你的账户里有余额，系统会自动用余额付掉

说实话，续费账单这个改成提前 7 天出，是个实用的改进。以前到期当天才出账单，时差一算要到第二天才能收到邮件，经常让人手忙脚乱。现在提前一周通知，处理起来从容多了。

直接说结论：**能，但不保证**。

实际情况是：大部分互联网公司、科技公司的财务认可这种 PDF Invoice，特别是你填好了公司抬头的情况下。
